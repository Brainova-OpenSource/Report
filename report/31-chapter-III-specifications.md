# Capítulo III: Requirements Specification
## 3.1. User Stories
# EPICS – BAKERYMANAGER (IoT)

| Epic ID | Nombre del Epic | Descripción |
|--------|----------------|-------------|
| EP-01 | Monitoreo IoT | Agrupa las funcionalidades relacionadas al monitoreo en tiempo real de sensores IoT en hornos, conservadoras, maquinaria y ambiente, permitiendo visualizar el estado operativo del negocio. |
| EP-02 | Alertas IoT | Incluye la generación de alertas automáticas cuando los sensores detectan valores fuera de rango, facilitando la detección temprana de fallas o riesgos operativos. |
| EP-03 | Notificaciones | Contempla el envío de notificaciones a los usuarios según su rol mediante distintos canales, asegurando la comunicación oportuna de eventos importantes del sistema. |
| EP-04 | Gestión de Datos y Registros | Permite almacenar, consultar y gestionar datos históricos de sensores, alertas, fallas y mantenimientos, asegurando trazabilidad y control de la información. |
| EP-05 | Reportes y Analítica | Incluye la generación de reportes, visualización de métricas, gráficos y análisis de datos para apoyar la toma de decisiones en el negocio. |
| EP-06 | Interfaz y Experiencia de Usuario | Agrupa las funcionalidades relacionadas con la interfaz del sistema, usabilidad, navegación, accesibilidad y experiencia del usuario en la plataforma. |
